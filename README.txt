== Description ==

Just install Plugin and enjoy feature of Clone Page Or Post Plugin.

This plugin used for clone page or post.

1). After activating plugin go to page or post list, below the page title options click on clone.

2). Whenever you click on clone duplicate copy of page is generated.

3). it's Done. Simple Right!!

== Installation ==
1. Upload the entire `clone-page-or-post` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the **Plugins** screen (**Plugins > Installed Plugins**).
